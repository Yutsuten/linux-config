*my-commands*

Custom commands:

`:Indent`             Set custom indent
`:Make`               Run `make` in the background
`:MakeLog`            Open the log of the last `:Make`
`:Pydoc`              Pydoc search word under cursor
`:ToggleFileInfo`     Toggle file information from the statusline
`:ToggleIndent`       Toggle 2-4 spaces indent

===============================================================

*my-shortcuts*

Custom shortcuts:

<leader>cc          Comment code
<leader>cu          Uncomment code
<leader>d           Go to definition
<leader>f           Find references for the symbol under cursor
<leader>e           Fix linter problems
<leader>h           Hover symbol under cursor
<leader>i           Toggle 2-4 spaces indent
<leader>n           Next lint warning/error
<leader>p           Previous lint warning/error
<leader>r           Run `:Make`
<leader>s           Search for symbols in the workspace
<leader>t           Rename current tab
<leader>%           Copy current directory into the clipboard
